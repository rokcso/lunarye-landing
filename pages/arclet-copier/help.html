Installation

Install Arclet Copier from the Chrome Web Store. The extension is free and requires minimal permissions to function.

Installation Steps:

  1. Visit the Chrome Web Store
  2. Click "Add to Chrome"
  3. Confirm the installation in the popup dialog
  4. The extension icon will appear in your toolbar

First Use

After installation, you can immediately start using Arclet Copier. Click the extension icon in your toolbar to open the interface.

Quick Tip: The extension automatically adapts to your preferred theme (light/dark) system settings!

Initial Setup

Configure your preferences in the extension settings. You can customize URL cleaning mode, theme, language, and copy behavior.

Recommended Settings:

  • URL Cleaning: Smart Mode
  • Theme: Follow System
  • Language: Your preferred language
  • Silent Copy Format: Markdown (for developers/writers)
Basic Features

üîó Copy URL

Copy the current page URL with your preferred cleaning mode applied.

How to Use:

  1. Navigate to any webpage
  2. Click the Arclet Copier extension icon
  3. Click the "Copy URL" button
  4. The cleaned URL is copied to your clipboard

Example:

Original: https://example.com/page?utm_source=google&utm_medium=cpc&utm_campaign=sale

Smart Mode: https://example.com/page

üìù Copy Markdown

Generate a Markdown link automatically using the page title and URL.

How to Use:

  1. Navigate to the page you want to link
  2. Click the extension icon
  3. Click "Copy Markdown"
  4. Paste the formatted link anywhere

Example Output:

[Example Page Title](https://example.com/page)

üì± Generate QR Code

Create a QR code for the current page URL for easy sharing across devices.

How to Use:

  1. Go to the page you want to share
  2. Click the extension icon
  3. Click "Generate QR Code"
  4. Scan the code with your mobile device

Tip: Perfect for sharing links in presentations or between devices!

URL Cleaning Modes

üîí Off Mode

Keep the original URL completely unchanged. No parameters are removed.

When to Use:

  • When you need all URL parameters for testing
  • When sharing with developers who need full URLs
  • For debugging purposes

Example:

Input: https://example.com/page?param1=value1&utm_source=google

Output: https://example.com/page?param1=value1&utm_source=google

üß† Smart Mode

Intelligently remove tracking and marketing parameters while preserving functional ones.

Removed Parameters:

  • utm_* (Google Analytics tracking)
  • fbclid (Facebook click ID)
  • gclid (Google click ID)
  • McDcid (Microsoft click ID)
  • And other known tracking parameters

Example:

Input: https://example.com/page?utm_source=google&user_id=123&fbclid=abc

Output: https://example.com/page?user_id=123

üßπ Aggressive Mode

Remove ALL URL parameters, leaving only the clean domain and path.

When to Use:

  • Maximum privacy and cleanliness
  • Academic and professional sharing
  • When parameters are not needed
  • Creating citations and references

Example:

Input: https://example.com/page?utm_source=google&user_id=123&token=abc

Output: https://example.com/page

Troubleshooting

üîß Extension Not Working

Issue:

Extension icon is grayed out or not responding.

Solution:

  1. Reload the current page
  2. Check if extensions are allowed on the current site
  3. Try clicking the extension icon again
  4. If on Chrome internal pages (chrome://*, etc.), this is normal behavior

üìã Copy Not Working

Issue:

URL or Markdown is not being copied to clipboard.

Solution:

  1. Check browser permissions for clipboard access
  2. Try copying manually to test clipboard functionality
  3. Reload the extension if the issue persists
  4. Some secure pages may restrict clipboard access

üåê URL Cleaning Issues

Issue:

URL parameters are not being cleaned as expected.

Solution:

  1. Verify your cleaning mode setting
  2. Check if the parameters are known tracking parameters
  3. Try "Aggressive Mode" for maximum cleaning
  4. Report the URL if you think certain parameters should be detected

üé≠ Theme Issues

Issue:

Theme is not switching or displaying correctly.

Solution:

  1. Check your system theme settings
  2. Try manually selecting a theme instead of "Follow System"
  3. Reload the extension popup
  4. Check if browser dark mode settings are interfering

Frequently Asked Questions

Q: Is my data private?

A: Yes! Arclet Copier processes everything locally and never sends any data to external servers. See our Privacy Policy for details.

Q: Can I add custom URL cleaning rules?

A: Currently, URL cleaning rules are predefined based on common tracking parameters. Custom rules may be added in future versions based on user feedback.

Q: How do I update the extension?

A: Chrome updates extensions automatically. You can check for updates manually in Chrome's extension management page.

Q: Can I use this on mobile devices?

A: Arclet Copier is designed for desktop Chrome. Mobile Chrome doesn't support the same extension API, but you can use the QR code feature to share links with mobile devices.

Q: What languages are supported?

A: Currently English and Simplified Chinese (ÁÆÄ‰Ωì‰∏≠Êñá) are fully supported. More languages may be added based on demand.

Q: How do I uninstall Arclet Copier?

A: Right-click the extension icon and select "Remove from Chrome" or go to Chrome's extensions page and click "Remove". All your local settings will be automatically deleted.
