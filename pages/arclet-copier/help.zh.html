智能URL复制Chrome扩展使用完整指南

快速入门
📥

安装

从Chrome网上应用店安装Arclet Copier。扩展是免费的，需要最小权限才能运行。

安装步骤：

  1. 访问 Chrome网上应用店
  2. 点击"添加到Chrome"
  3. 在弹出对话框中确认安装
  4. 扩展图标将出现在工具栏中
🚀

首次使用

安装后，您可以立即开始使用Arclet Copier。点击工具栏中的扩展图标打开界面。

快速提示：扩展会自动适应您偏好的主题（明/暗）系统设置！

⚙️

初始设置

在扩展设置中配置您的偏好。您可以自定义URL清理模式、主题、语言和复制行为。

推荐设置：

  • URL清理：智能模式
  • 主题：跟随系统
  • 语言：您的偏好语言
  • 静默复制格式：Markdown（适合开发者/作家）
基础功能

🔗 复制URL

使用您偏好的清理模式复制当前页面URL。

使用方法：

  1. 导航到任何网页
  2. 点击Arclet Copier扩展图标
  3. 点击"复制URL"按钮
  4. 清理后的URL已复制到剪贴板

示例：

原始： https://example.com/page?utm_source=google&utm_medium=cpc&utm_campaign=sale

智能模式： https://example.com/page

📝 复制Markdown

使用页面标题和URL自动生成Markdown链接。

使用方法：

  1. 导航到要链接的页面
  2. 点击扩展图标
  3. 点击"复制Markdown"
  4. 在任何地方粘贴格式化链接

输出示例：

[示例页面标题](https://example.com/page)

📱 生成二维码

为当前页面URL创建二维码，便于在设备间分享。

使用方法：

  1. 进入要分享的页面
  2. 点击扩展图标
  3. 点击"生成二维码"
  4. 用移动设备扫描二维码

提示： 在演示中或设备间分享链接时完美适用！

URL清理模式

🔒 关闭模式

保持原始URL完全不变。不删除任何参数。

使用时机：

  • 当您需要所有URL参数进行测试时
  • 当需要向开发者提供完整URL时
  • 用于调试目的

示例：

输入: https://example.com/page?param1=value1&utm_source=google

输出: https://example.com/page?param1=value1&utm_source=google

🧠 智能模式

智能移除跟踪和营销参数，同时保留功能性参数。

删除参数：

  • utm_* (Google Analytics跟踪)
  • fbclid (Facebook点击ID)
  • gclid (Google点击ID)
  • McDcid (Microsoft点击ID)
  • 以及其他已知跟踪参数

示例：

输入: https://example.com/page?utm_source=google&user_id=123&fbclid=abc

输出: https://example.com/page?user_id=123

🧹 激进模式

删除所有URL参数，仅保留清洁的域名和路径。

使用时机：

  • 最大隐私和清洁度
  • 学术和专业分享
  • 当不需要参数时
  • 创建引用和引文时

示例：

输入: https://example.com/page?utm_source=google&user_id=123&token=abc

输出: https://example.com/page

故障排除

🔧 扩展不工作

问题：

扩展图标呈灰色显示或无响应。

解决方案：

  1. 重新加载当前页面
  2. 检查当前站点是否允许使用扩展
  3. 再次尝试点击扩展图标
  4. 如果在Chrome内部页面（chrome://*等），这是正常行为

📋 复制不工作

问题：

URL或Markdown未被复制到剪贴板。

解决方案：

  1. 检查剪贴板访问的浏览器权限
  2. 手动测试复制功能以测试剪贴板是否正常
  3. 如果问题持续，重新加载扩展
  4. 某些安全页面可能限制剪贴板访问

🌐 URL清理问题

问题：

URL参数未按预期清理。

解决方案：

  1. 验证您的清理模式设置
  2. 检查参数是否为已知跟踪参数
  3. 尝试"激进模式"以获得最大清理
  4. 如果您认为某些参数应该被检测到，请报告该URL

🎭 主题问题

问题：

主题未切换或显示不正确。

解决方案：

  1. 检查您的系统主题设置
  2. 尝试手动选择主题而不是"跟随系统"
  3. 重新加载扩展弹出窗口
  4. 检查浏览器暗色模式设置是否有干扰

常见问题

问：我的数据是私有的吗？

答： 是的！Arclet Copier在本地处理所有内容，从不向外部服务器发送任何数据。有关详细信息，请参阅我们的隐私政策。

问：我可以添加自定义URL清理规则吗？

答： 目前，URL清理规则是基于常见跟踪参数预定义的。根据用户反馈，可能会在未来版本中添加自定义规则。

问：如何更新扩展？

答： Chrome会自动更新扩展。您可以在Chrome的扩展管理页面手动检查更新。

问：我可以在移动设备上使用吗？

答： Arclet Copier专为桌面Chrome设计。移动Chrome不支持相同的扩展API，但您可以使用二维码功能与移动设备分享链接。

问：支持哪些语言？

答： 目前完全支持英语和简体中文。可根据需求添加更多语言。

问：如何卸载Arclet Copier？

答： 右键点击扩展图标并选择"从Chrome中删除"，或转到Chrome的扩展页面并点击"删除"。所有本地设置将自动删除。
